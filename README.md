# LeetCode Practice (Java)

Collection of Java problem-solving practice covering arrays, binary search, recursion, backtracking, sorting, and OOP basics.

## Goals

- Build consistency in data structures and algorithms practice.
- Keep solutions grouped by topic for fast review before interviews.
- Track improvement over time with clear, runnable source files.

## Tech Stack

- Language: Java
- IDE: IntelliJ IDEA (optional)

## Repository Structure

```text
src/
├── Arrays/
├── BackTracking/
├── BinarySearch/
├── Easy/
├── OOPS/
├── PractisePlease/
├── Recursion/
├── Sort/
├── SortingRecursion/
├── StringRecursion/
├── Strings/
├── patterns/
└── ... (additional standalone practice files)
```

## Run A Solution

1. Open the repo in IntelliJ and run any class with a `main` method.
2. Or run from terminal:

```bash
javac src/BinarySearch/BinarySearch.java
java -cp src BinarySearch.BinarySearch
```

Adjust the class path/package based on the target file.

## Practice Conventions

- Keep one problem per file whenever possible.
- Add a short comment at top of each file for problem name/link.
- Prefer descriptive file names and package folders by topic.
- Do not commit generated/IDE files.
